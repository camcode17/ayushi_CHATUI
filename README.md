# Chat UI Assignment

## Description
A responsive ChatGPT-like chat interface built using HTML, CSS, JavaScript, jQuery, and Bootstrap 5.

## Features
- Chat UI with user & AI messages
- Typing indicator animation
- Suggestion cards
- Auto-resizing textarea
- Sidebar with chat history
- Mobile responsive design
- Dark mode toggle

## How to Run
1. Open index.html in the browser
2. Start chatting!

## Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap 5
